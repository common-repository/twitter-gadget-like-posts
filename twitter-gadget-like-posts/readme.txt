This plug-in is the ability to display a post of a feeling similar to Gadget of Twitter

== Description ==

* This plug-in is the ability to display a post of a feeling similar to Gadget of Twitter
* You can select more than one category of post to display is possible.
* You can specify the background color, border color, link color, text color to suit your theme I am available.
* It is also possible to specify the height of the display area.
* You can also specify whether to display the fact that many characters from the beginning of one post.
* You can specify the number of articles I am available to be displayed.
* You can also change the display speed.

â– Initial value
* all height = 500px (max=1000px min=100px)
* post number = 10
* display interval = 5 sec (max=10 min=1)
* 1 post length = 300
* link_color = #0000FF
* color = #000000
* bg_color = #FFFFFF
* border_color = #EFEFEF

== Installation ==

1. Upload `twitter-gadget-like-posts` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Please be placed in the sidebar by dragging and dropping the "twitter gadget like posts" in the menu 'Widget'
